# facepaint

facepaint is an AR drawing app where users can sign up, create hand-drawn face filters, try on filters, save favorites, and browse other users' work.

![A GIF sample of a filter being drawn.](https://i.imgur.com/3u574f8.gif)

This repository contains two apps:

- [facepaint fron end](facepaint-frontend/README.md)
- [facepaint backend](facepaint-backend/README.md)

For local development, start the backend first then start the frontend.
