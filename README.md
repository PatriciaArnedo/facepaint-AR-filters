# facepaint

facepaint is an AR drawing app where users can sign up, create hand-drawn face filters, try on filters, save favorites, and browse other users' work.

This repository contains two apps:

- Frontend: [`facepaint-frontend/README.md`](/Users/patricia/Code/facepaint/facepaint-frontend/README.md)
- Backend: [`facepaint-backend/README.md`](/Users/patricia/Code/facepaint/facepaint-backend/README.md)

For local development, start the backend first and then start the frontend.
